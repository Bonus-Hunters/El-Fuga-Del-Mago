# El Fuga del Mago

**El Fuga del Mago** is a first-person action-adventure RPG designed to challenge the player's reflexes, strategy, and ability to adapt in a hostile world. Merging fast-paced combat with exploration and survival elements, the game creates a thrilling experience inside a mysterious virtual realm where the only rule is to survive.

Sherlock is warned by a goblin that death in this realm means death in reality . To escape, he must beat the game from the inside, uncovering secrets and confronting the hidden force behind the trap.

## Technical & Programming Features

* **Dynamic Dialogue System:** The game features a dynamic dialogue system integrated directly with the quest mechanics. This structure allows developers to easily modify dialogues and apply them across multiple quests.
* **Scalable Architecture:** Both the quest and combat systems are built to be dynamic and easy to scale up, facilitating the addition of new features and bigger quests.
* **Modular Asset Implementation:** The development workflow is streamlined using ready-made prefabs that are designed to be simply dragged and dropped into the environment.
